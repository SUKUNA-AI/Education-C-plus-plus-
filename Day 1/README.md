# C++ Learning Journey — День 1

## Обзор

В этом репозитории находятся мои решения за **первый день** изучения **C++20**.

Цель этого этапа — не писать большой проект слишком рано, а **укрепить базу языка** через несколько маленьких самостоятельных консольных программ.

Темы первого дня:
- ветвления
- циклы
- функции
- базовый ввод и вывод
- простая математическая логика

---

## Выполненные задачи

### 1. `calculator.cpp`
Простой калькулятор на `switch`.

Поддерживаемые операции:
- сложение
- вычитание
- умножение
- деление

Что отрабатывается:
- `switch`
- ввод данных
- обработка некорректной операции
- проверка деления на ноль

---

### 2. `even_or_odd.cpp`
Проверка числа на чётность или нечётность.

Что отрабатывается:
- `if / else`
- оператор `%`
- базовый ввод/вывод для целых чисел

---

### 3. `multiplication_table.cpp`
Вывод таблицы умножения.

Что отрабатывается:
- вложенные циклы
- форматирование вывода
- понимание структуры «цикл внутри цикла»

---

### 4. `prime_check.cpp`
Проверка числа на простоту.

Что отрабатывается:
- вынесение логики в функцию
- циклы и условия
- базовая оптимизация проверки делителей

---

### 5. `factorial.cpp`
Вычисление факториала числа.

Что отрабатывается:
- отдельная функция для вычисления
- итеративная логика
- возврат значения из функции
- накопление результата

---

### 6. `gcd_lcm.cpp`
Вычисление НОД и НОК двух чисел.

Что отрабатывается:
- несколько функций в одной программе
- алгоритм Евклида
- переиспользование одной функции внутри другой

## Слабые места и что улучшать

### 1. Проверка ввода
Некоторые программы можно улучшить за счёт более аккуратной обработки некорректного ввода.

### 2. Граничные случаи
Нужно внимательнее относиться к случаям:
- отрицательных чисел
- нуля
- некорректного пользовательского ввода

### 3. Проектирование функций
Некоторые функции можно сделать чище: аккуратнее выбирать передачу параметров и упрощать интерфейс.

### 4. Единый стиль
Стоит придерживаться одного стиля ввода/вывода и более аккуратного оформления кода во всех файлах.

## Сборка и запуск

Любой файл можно собрать через `g++` с поддержкой C++20.

Пример:

```bash
g++ -std=c++20 calculator.cpp -o calculator
./calculator
```

Остальные файлы:

```bash
g++ -std=c++20 even_or_odd.cpp -o even_or_odd
g++ -std=c++20 multiplication_table.cpp -o multiplication_table
g++ -std=c++20 prime_check.cpp -o prime_check
g++ -std=c++20 factorial.cpp -o factorial
g++ -std=c++20 gcd_lcm.cpp -o gcd_lcm
```
